Folashade Ayodele Key points from today's class 1. Understanding the distinction between Git and GitHub. 2. Initializing a Git repository with the `git init` command. 3. Staging changes using `git add .` to include all files. 4. Committing changes with `git commit -m "commit message"`. 5. Viewing the commit history using `git log`. 6. Connecting to a remote repository with `git remote add origin `. 7. Pushing changes to a remote branch using `git push origin `. 8. Pulling updates from a remote branch using `git pull origin `.
